LARPs you can run

The table below contains the list of LARPs currently available in the NEIL LARP Game Bank. NEIL welcomes new submissions to the Game Bank, to give other LARPers a chance to experience your LARP.

  • Clicking on the header for a column will sort the table by that column.

  • Clicking on the link on the name of the LARP will take you to a page with much more detail about the LARP. The detailed page contains the link you need to download that LARP's materials.

  • Clicking on the link in the License column takes you to a page that provides the specifics of the licensing arrangement for the LARP. This will tell you if you are allowed to charge money to run the LARP, or if you can make changes to the LARP, or what other restrictions need to be followed to run the LARP.

These LARPs represent the inspirations and hard work by some very talented authors. They've taken the extra step of making their work available to you to run. Please respect that generosity by crediting the authors with their work and by following the licensing model they've selected. Enjoy them.
