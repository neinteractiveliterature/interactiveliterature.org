Writing a Good Blurb for your Submission

  • It has to tell me about the setting / style / tone / genre / kinds of characters of the game
  • It has to tell me what my character might be doing during the game - it has to set character expectations
  • It has to tell me what I will be doing as a player in the game - it has to set player expectations
  • It has to sell me and get me interested in playing, as a character and a player
